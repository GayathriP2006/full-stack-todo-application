# Full Stack To-Do Application

## Overview

A Full Stack To-Do Application built using React, Node.js, Express.js, and MongoDB Atlas. Users can manage tasks with complete CRUD operations and navigate through authentication pages.

## Features

* User Login Page
* User Registration Page
* Task Creation
* View All Tasks
* Update Task Status
* Delete Tasks
* React Routing
* MongoDB Atlas Integration
* Responsive User Interface

## Technologies Used

### Frontend

* React.js
* React Router DOM
* Axios

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## API Endpoints

### Create Task

POST /tasks

### Get Tasks

GET /tasks

### Update Task

PUT /tasks/:id

### Delete Task

DELETE /tasks/:id

## Installation

### Backend

npm install
node server.js

### Frontend

cd client
npm install
npm start
